Installation in your app

Step 1 Install Squatch.js

Add the following JavasSript at the end of your <body> tag on every page in your application, and replace the data fields with information about the logged-in user.

<script type="text/javascript">
    var _sqh = _sqh || [];
    
    // Push user details for a user who is logged in to your system
    _sqh.push(['init', {
        tenant_alias: '$tenant_alias', // REPLACE: Your live or test mode tenant_alias. This identifies your app.
        account_id: '$accountId', // REPLACE: the Stripe/Recurly/Braintree/Zuora id used for payment system integration
        user_id: '$userId', // REPLACE: A user ID from your system (must be unique for every user)
        email: '$emailAddress', // REPLACE: The e-mail address of the user
        first_name: '$firstName', // REPLACE: The user's first name
        last_name: '$lastName', // (OPTIONAL) REPLACE: The user's last name
        user_image: '$userImageURL', // (OPTIONAL) REPLACE: Absolute URL to the user's profile image
        fb_share_image: '$facebookImageURL' // (OPTIONAL) REPLACE: Absolute URL to the image included when users refer on Facebook
    }]);
    
    // Include squatch.js
    (function(){function l(){var s=document.createElement("script");s.type="text/javascript";s.async=true;s.src=location.protocol+"//d2rcp9ak152ke1.cloudfront.net/assets/javascripts/squatch.min.js";var t=document.getElementsByTagName("script")[0];t.parentNode.insertBefore(s,t)}if(window.attachEvent){window.attachEvent("onload",l)}else{window.addEventListener("load",l,false)}})();
    </script>

See also our init function reference and our Page Speed Best Practice Guide.

Step 2 Add a button

Add the squatchpop class to any button on your page that you want to open the referral program widget.

Examples:

<button class="squatchpop">Referral Discount</button> <!-- Clicking this opens the referral program in a modal popup -->
<a href="#" class="squatchpop">Invite a Friend</a> <!-- Clicking this would also open the popup -->
<div class="btn squatchpop">Love our service? Share the wealth!</div> <!-- Yep, DIVs work, too. Put 'em everywhere. -->

You can put lots of buttons and links on a page. Remember to make these buttons prominent and easy to find, otherwise people might not know you have a referral program. The referral widget is shown as a JavaScript popup, so it’s safe to include these buttons on every page. See also our Refer-a-friend Buttons Best Practice Guide for ideas on how to boost participation in your program.

Step 3 Autofill the Coupon Code

Add a call to autofill the coupon code on your payment page. This reduces the chance of someone not using their referral discount when they sign up.

Example:

<input type="text" name="Coupon" id="my_coupon" value="" />
_sqh.push(['autofill', '#my_coupon']);

Remember to still let people manually enter their coupon codes on your payment page in case they were referred face-to-face -- that’s why we create easy-to-remember coupon codes based off of people's names like JESSICAALBA and ISAACNEWTON.

See also our autofill function reference for a more complete example.
