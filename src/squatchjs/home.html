Squatch.js Version 2

Squatch.js Version 2 is based on the latest Javascript standards. Version 2 has functionality to let you make REST API calls, embed widgets, and track referrals. It is modular, works in both the browser and node, and compatible with libraries such as React, Webpack and Browserify.

  • Version 2 Quickstart
  • Version 2 API Reference
Example 
window.squatch.ready(function(){
            
    squatch.init({
        tenantAlias: 'test_bpinhag9yagag'
    });
    
    squatch.api.createUser({
        id: '5678',
        accountId: 'abc',
        email: 'bob@example.com',
        firstName: 'Bob'
    });

});

Squatch.js Version 1

Squatch.js Version 1 is an asynchronous tracking library similar to Google Analytics. It uses the global window._sqh queue for asynchronous API calls. Version 1 has a limited API and can only be used to embed a single widget on a page.

  • Version 1 Quickstart
  • Version 1 API Reference
Example 
window._sqh = window._sqh | [];
            
_sqh.push(['init',{
    tenant_alias: 'test_bpinhag9yagag',
    user_id: '5678',
    account_id: 'abc',
    email: 'bob@example.com',
    first_name: 'Bob'
});

Which one?

We recommend that new sites use Squatch.js Version 2. If you started using SaaSquatch before October 2016, you may be using Version 1, and you might want to upgrade.

Which one am I already using?
  • window.squatch? - you're using Version 2
  • window._sqh? - you're using Version 1
