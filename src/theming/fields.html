--- title: Theme Field Reference highlights: Your handlebar template files are provided with variables called a "context". This reference explains each of these fields. slug: themes/fields template: themes.html ---

Fields

tenantAlias The active tenant
user The currently logged in user details
id A unique identifier
accountId The account this user is part of
referralCode The referral code for sharing
email A contact email (or null)
imageUrl An href URL for the use (or null)
firstName The user's first name
lastName The user's last name (or null)
referrals An array of referrals for this user
referralsLength An integer length of the referrals array
rewardBalances An array of reward balances for this user
shareLinks A map of share links with various tags
mode The mode the widget has been loaded in
programDetails An object describing the configured program
